# Folder sharing application

## Description

This project is a web application(only desktop verion) built with TypeScript, React, and Styled Components. It allows users to create folders and share them through generated links. The backend for this project can be found on [GitHub](link-to-backend-repository).

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)
- [Contributing](#contributing)
- [License](#license)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repository.git
   ```

2. Navigate to the project directory:

   ```bash
   cd your-repository
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## Usage

To start the application, run:

```bash
npm start
```

This will launch the application in development mode. Open [http://localhost:3000](http://localhost:3000) in your browser to view it.

## Features

- Create folders with a user-friendly interface.
- Share folders by generating and sharing unique links.
